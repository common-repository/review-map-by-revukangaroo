== Description ==
Show off your customer's online reviews with Review Map by Revukangaroo. This plugin is an added upgrade to the Customer Review plugin by Revukangaroo.

Your reviews will instantly appear on your website with geo-targeted locations for each. Your customers will see the average reviews of each of your employees.

Enjoy all of the SEO benefits with fresh new content that is placed on your review pages. Our API ties direct into Google Maps.

== Installation ==
1. Upload the review-map-by-revuKangaroo folder to the /wp-content/plugins/ directory.
2. Activate the Review Map by RevuKangaroo plugin through the \\\'Plugins\\\' menu in WordPress.
3. Go to Settings => Schema API Settings and  submit your API Key and update other settings there.
4. Create a new page and assign RevuKangaroo Review Map Template to that page, for the cities pages please assign this page as parent of those pages which have been created for Cities.
